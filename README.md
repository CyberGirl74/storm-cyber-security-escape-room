# Storm's Cyber & Data: Escape Room ⚡🌩️

## Project Overview

Storm's Cyber & Data: Escape Room is an educational cybersecurity game being developed in GDevelop.

This project combines cybersecurity concepts, problem-solving, and interactive gameplay while documenting my journey as a Cyber & Data Security Technology student.

## Project Goals

- Introduce cybersecurity concepts through gameplay
- Create engaging challenges and puzzles
- Develop problem-solving skills
- Learn game development using GDevelop
- Build a portfolio project while learning cybersecurity

## Future Vision

Storm's Cyber & Data: Escape Room is being designed as an educational cybersecurity experience that teaches real-world concepts through gameplay.

Players will solve security challenges, investigate incidents, and learn cybersecurity principles while progressing through interactive missions.

## Learning Journey

This project is part of my transition from a 20-year career as an Ironworker and Welder into Cyber & Data Security Technology.

As I continue learning cybersecurity, networking, Linux, Python, Wireshark, and Splunk, this project allows me to apply creativity while building technical skills.
